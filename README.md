# TicTac Pro

TicTac Pro is a browser-based implementation of the classic Tic Tac Toe game built with HTML, CSS, and JavaScript. The project combines responsive design, interactive animations, and multiple gameplay modes to create a polished web gaming experience.

## Overview

The application supports both local multiplayer and single-player gameplay against an AI opponent with multiple difficulty levels. It focuses on responsive design, smooth gameplay, and a lightweight architecture without requiring a backend.

## Features

* Player vs Player mode
* Player vs AI mode
* Three AI difficulty levels
* Win and draw detection
* Persistent score tracking using Local Storage
* Glassmorphism-inspired user interface
* Responsive design for desktop and mobile devices
* Confetti animation on victory

## Technology Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Canvas Confetti
* Local Storage API
